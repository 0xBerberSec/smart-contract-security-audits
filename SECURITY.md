# Security Policy

This repository is a public security research portfolio.

## Reporting Issues

If you find an inaccuracy, broken link, reproducibility issue, or unsafe claim in a report, please open a GitHub issue or pull request.

## Scope

This repository contains:

- Public audit notes
- QA findings
- Proof-of-concept tests
- Security research writeups

It does not operate production systems or hold user funds.

## Responsible Disclosure

Do not publish private exploit details for live protocols in this repository unless they have already been disclosed, remediated, or approved for publication.
